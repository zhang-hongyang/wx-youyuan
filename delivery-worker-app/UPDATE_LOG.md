# 功能更新日志

## v1.1.0 更新内容（2024-10-28）

### 1. 行程定位显示功能 ✅
**功能说明**：送货员可实时共享位置给客户查看

**页面路径**：`pages/tracking/tracking`

**主要特性**：
- 实时地图显示送货员当前位置
- 预计到达时间（ETA）自动计算
- 剩余距离显示
- 配送进度时间线
- 一键联系送货员
- 支持司机端分享/停止共享位置

**使用方式**：
- 司机端：从送货单详情页点击"行程共享"开启位置共享
- 客户端：通过分享的链接或订单详情查看实时位置

### 2. 报账功能 ✅
**功能说明**：送货员可上传账单截图和发票申请报销

**页面路径**：`pages/reimbursement/reimbursement`

**报销规则**：
| 费用类型 | 需要发票 | 说明 |
|---------|---------|------|
| 在外就餐 | ❌ 免发票 | 凭账单截图即可报销 |
| 停车费 | ✅ 需发票 | 必须上传发票照片 |
| 过路费 | ✅ 需发票 | 必须上传发票照片 |
| 加油费 | ✅ 需发票 | 必须上传发票照片 |
| 维修费 | ✅ 需发票 | 必须上传发票照片 |
| 其他费用 | ✅ 需发票 | 必须上传发票照片 |

**主要特性**：
- 关联订单选择
- 费用类型选择（带发票要求标识）
- 发票照片上传（最多4张）
- 账单/支付截图上传
- 其他凭证上传
- 实时费用预估
- 报销历史记录

### 3. 效率超时人工判断 ✅
**功能说明**：根据订单规模由管理人员判断超时是否合理

**订单规模分类**：
| 规模 | 货物数量 | 预计用时 | 标识 |
|-----|---------|---------|------|
| 小型订单 | 1-5件 | ≤2小时 | 🟢 绿色 |
| 中型订单 | 6-12件 | 2-4小时 | 🟡 黄色 |
| 大型订单 | >12件 | >4小时 | 🔴 红色 |

**超时审核流程**：
1. 系统根据订单规模给出预计用时
2. 送货员实际完成时间超过预计用时的1.3倍时标记为超时
3. 送货员可提交超时原因
4. 管理人员人工审核超时合理性
5. 审核通过则不扣绩效分，驳回则按超时处理

**相关页面**：
- `pages/delivery-detail/delivery-detail` - 显示超时审核状态
- 绩效统计页面 - 显示审核后的超时记录

### 4. 临时工申请系统 ✅
**功能说明**：送货员可申请临时工配合，临时工下班必须拍照打卡

#### 4.1 临时工申请
**页面路径**：`pages/temp-worker-request/temp-worker-request`

**申请流程**：
1. 选择需要配合的订单
2. 设置申请人数（1-5人，系统根据订单规模给出建议）
3. 选择预计工作时长（2/4/6/8小时或自定义）
4. 选择申请原因（货物较多/大件搬运/紧急订单/人手不足/其他）
5. 填写工作安排说明（选填）
6. 查看费用预估（¥30/小时/人）
7. 提交申请等待审核

**特性**：
- 智能人数建议（根据订单货物数量）
- 实时费用预估
- 申请历史记录

#### 4.2 临时工下班打卡
**页面路径**：`pages/temp-check-out/temp-check-out`

**打卡要求**：
- ✅ 定位确认（必须在客户现场范围内）
- ✅ 工作确认（完成任务/归还工具/清理现场）
- ✅ 现场拍照（至少1张工作成果照片）
- ✅ 负责人签字确认

**结算规则**：
- 按实际工作时长结算
- 单价：¥30/小时
- 费用 = 人数 × 时长 × 单价

**示例**：
- 2人工作4小时 = 2 × 4 × 30 = ¥240

## 使用指南

### 对于正式员工：
1. **申请临时工**：工作台 → 申请临时工 → 选择订单 → 填写申请
2. **费用报销**：工作台 → 费用报销 → 选择订单 → 上传发票/账单
3. **行程共享**：送货单详情 → 行程共享 → 开启位置共享
4. **查看超时审核**：送货单详情 → 查看超时判定状态

### 对于临时工：
1. 收到工作安排通知后按时到岗
2. 工作完成后进行**下班打卡**
3. 必须完成：拍照 → 工作确认 → 负责人签字
4. 等待审核通过后结算工资

### 对于管理人员：
1. **审核临时工申请**：管理后台 → 临时工申请审核
2. **审核超时申诉**：管理后台 → 超时申诉审核
3. **确认临时工下班**：现场签字确认临时工打卡
4. **审核报销单据**：管理后台 → 报销审核

## 注意事项

1. **行程定位**：开启后会每30秒更新一次位置，注意手机电量
2. **发票要求**：除就餐外所有费用类型必须上传发票，否则不予报销
3. **临时工打卡**：临时工必须完成下班打卡才能结算工资，请务必现场监督并签字确认
4. **超时申诉**：如有合理超时原因，请在完工后24小时内提交申诉
