# 库管系统功能说明

## 功能概述

库管系统是小程序新增的管理模块，提供订单排班、人员车辆安排、验收管理、维修管理、统计报表等功能。

## 功能模块

### 1. 订单排班表 (pages/schedule)
**功能**：以课表形式展示一周订单安排

**主要特性**：
- 周视图展示，类似大学生课表
- 7天 × 17个时间段（6:00-22:00）
- 订单状态标识（已安排/待安排/临时单/紧急）
- 点击订单查看详情
- 底部待处理订单列表
- 支持切换周、回到今天

**使用流程**：
1. 库管登录后进入排班表
2. 查看待处理订单列表
3. 点击订单进行人员车辆安排
4. 安排后系统自动通知员工

---

### 2. 订单编辑/人员车辆安排 (pages/order-edit)
**功能**：为订单安排人员、车辆、物资

**安排内容**：
| 类别 | 选项 | 说明 |
|-----|------|------|
| **人员** | 正式员工 | 显示技能标签、今日任务数、空闲状态 |
| | 临时工 | 显示联系电话、空闲状态 |
| **车辆** | 自有车辆 | 显示车牌、车型、载重、空闲状态 |
| | 货拉拉 | 第三方叫车 |
| | 滴滴货运 | 第三方叫车 |
| | 其他 | 手动输入车辆信息 |
| **物资** | 板车 | 设置借用数量，实时显示库存 |
| | 运输笼 | 设置借用数量，实时显示库存 |
| **时间** | 装车时间 | 15分钟~2小时 |
| | 路途时间 | 15分钟~2小时 |
| | 布置时间 | 1小时~5小时 |

**使用流程**：
1. 选择负责人员（可多选）
2. 选择运输车辆
3. 配置物资数量
4. 确认时间预算
5. 提交后自动通知员工

---

### 3. 临时订单上传 (pages/temp-order-upload)
**功能**：快速上传临时性订单

**特点**：
- 立即通知所有员工
- 支持设置紧急程度（普通/紧急/立即）
- 快速添加货物清单
- 自动发送推送通知

---

### 4. 验收管理 (pages/inspection)
**功能**：订单完成后验收，统计丢失和损坏

**验收内容**：
1. **货物清点**
   - 对比应收/实收数量
   - 自动计算缺失数量
   - 标记异常物品

2. **物资归还**
   - 板车数量核对
   - 运输笼数量核对

3. **损坏登记**
   - 分类：报废 / 维修
   - 拍照记录
   - 关联到具体订单和员工

4. **自动统计**
   - 缺失总数
   - 报废总数
   - 维修总数
   - 正常归还数

**同步机制**：
- 验收结果同步到员工绩效记录
- 损坏数量计入员工考核
- 维修物品进入维修队列

---

### 5. 维修管理 (pages/repair-manage)
**功能**：管理损坏物品的维修流程

**状态分类**：
- 待维修：新登记的损坏物品
- 维修中：已安排维修的物品
- 已完成：维修完成的物品

**提醒功能**：
- 今日维修完成提醒
- 维修进度追踪
- 预计完成时间

---

### 6. 统计报表 (pages/warehouse-stats)
**功能**：损坏统计和库存统计

**损坏统计**：
- 报废总数
- 维修总数
- 损失金额
- 损坏趋势折线图（近30天）
- 损坏类型分布（饼图）
- 易损物品排行

**库存统计**：
- 板车库存
- 运输笼库存
- 其他物资库存
- 库存变化趋势

---

### 7. 库管工作台 (pages/warehouse)
**功能**：库管首页，一站式管理入口

**显示内容**：
- 今日订单数
- 待验收数量
- 待维修数量
- 库存不足预警
- 快捷操作入口
- 今日订单列表
- 库存预警列表

---

## 数据流转

```
业务员上传订单
    ↓
库管排班安排 → 选择人员/车辆/物资/时间
    ↓
系统自动通知员工
    ↓
员工执行任务
    ↓
任务完成 → 库管验收
    ↓
统计丢失/损坏
    ↓
├─→ 正常 → 结束
├─→ 丢失 → 计入员工绩效
└─→ 损坏 → 分类处理
    ├─→ 报废 → 更新库存
    └─→ 维修 → 进入维修队列
        ↓
    维修完成 → 更新库存
```

## 权限控制

库管系统仅对 `userInfo.role === 'warehouse'` 的用户显示

首页添加入口判断：
```xml
<view class="card" wx:if="{{userInfo.role === 'warehouse'}}">
  <!-- 库管系统入口 -->
</view>
```

## 使用说明

### 库管人员
1. 每日查看排班表，安排新订单
2. 上传临时订单（如有）
3. 订单完成后进行验收
4. 定期检查库存预警
5. 跟进维修进度

### 送货员工
1. 接收订单通知
2. 按安排执行任务
3. 任务完成后等待验收
4. 查看验收结果（同步到绩效）

## 注意事项

1. **物资配备**：每个订单必须配备板车和运输笼，用于验收时核对
2. **临时订单**：会立即通知所有员工，请谨慎发布
3. **损坏分类**：报废和维修的区别影响库存计算和成本统计
4. **维修提醒**：系统会在维修预计完成日发送提醒
