# 最小回归清单（云开发版）

更新时间：2026-03-07

## 执行前提

1. 小程序可正常启动
2. 已在微信开发者工具部署云函数 api
3. 当前云环境可用
4. 已按需设置 runtimeSettings

---

## A. 登录链路

- [ ] 微信登录：可进入登录流程并拿到微信身份
- [ ] 首次绑定：输入 employeeId + bindCode 可绑定账号
- [ ] 已绑定用户再次登录可直接进入对应角色入口
- [ ] 登录后 userInfo 和 token 写入本地存储
- [ ] 非法绑定码或不存在员工编号时给出明确错误提示

## A1. 角色入口与页面守卫

- [ ] 管理者登录后可进入管理入口页面
- [ ] 执行者登录后可进入执行工作台
- [ ] 财务登录后可进入报销审核入口
- [ ] 非授权角色访问管理/财务页面时被拦截

## B. 工作台与任务

- [ ] 首页可从云函数加载今日任务列表
- [ ] 有进行中订单时显示当前订单
- [ ] 点击任务可进入订单详情

## C. 打卡链路

- [ ] 未上班打卡时，到达/完工提示受限
- [ ] 上班打卡可提交到云端并同步写入本地记录
- [ ] 到达打卡可提交并关联订单
- [ ] 完工打卡可提交并关联订单

## D. 报销链路

- [ ] 可选择订单与费用类型
- [ ] 需发票类型未上传发票时被拦截
- [ ] 上传图片后返回云文件 fileID
- [ ] 提交成功后可在历史中看到记录

## E. 问题上报链路

- [ ] 问题类型选择正确
- [ ] 关键必填项校验生效
- [ ] 图片可上传到云存储
- [ ] 提交后返回上一页或提示成功
- [ ] 安全事故提交显示紧急提示

## F. 云开发专项

- [ ] allowLocalFallback=false 且云环境可用时，登录/任务/打卡成功
- [ ] allowLocalFallback=false 且云函数报错时，界面显示错误
- [ ] allowLocalFallback=true 时，云端失败后页面可回退本地数据
- [ ] 云函数 api 已为当前环境部署最新代码
- [ ] 高风险接口越权调用被拒绝（如管理员删除员工、财务审核）
